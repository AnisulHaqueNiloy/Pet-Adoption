# Pet Adoption Website

    This is a pet adoption website named Peddy. In this site anyone can adopt their favourite pets. We provide all kinds of categories of pets. Before you adopted you can see the pet details briefly. We also sorted the animal according to their price.

## Key Features

The key features of this project are given below -

- 1.Price Sorting: Sort pets by price in descending order for easy comparison.

- 2.Category Display: Browse pets organized by various categories.

- 3.Loading Spinner: Visual feedback with a spinner while fetching data.

- 4.Like Button: Easily add pet photos to a "Liked" section for later viewing.

- 5.Adopt and Details Buttons: Initiate the adoption process and view detailed pet information in a modal.
